    Remove-PodeTimer

    SYNOPSIS

    Removes a specific Timer.

    SYNTAX

    Remove-PodeTimer [-Name] <String> [<CommonParameters>]

    DESCRIPTION

    Removes a specific Timer.

    EXAMPLES

    EXAMPLE 1

    Remove-PodeTimer -Name 'SaveState'

    PARAMETERS

    -Name

    The Name of Timer to be removed.

    Type: String
Parameter Sets: (All)
Aliases:

Required: True
Position: 1
Default value: None
Accept pipeline input: True (ByValue)
Accept wildcard characters: False

    CommonParameters
